# OpenGrid

> A distributed, peer‑to‑peer LLM inference network powered by volunteer compute and token credits

---

## What this is

This repository contains the complete architectural specification for OpenGrid, a permissionless, volunteer‑powered inference network that allows anyone with a consumer PC, gaming GPU, or multi‑core CPU to contribute compute time and earn inference credits in return.
